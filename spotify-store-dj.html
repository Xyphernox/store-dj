Step 01

Spotify API credentials

Get these from the Spotify Developer Dashboard. This app uses Authorization Code + PKCE — Client Secret is not needed.

Before clicking Connect:
1. In Terminal: cd /Users/dris/Desktop/Claude/Store_DJ && python3 -m http.server 8888
2. Open http://127.0.0.1:8888/spotify-store-dj.html in your browser
3. In your Spotify app settings, add Redirect URI exactly: http://127.0.0.1:8888/spotify-store-dj.html — click Add, then Save

Scopes: playlist-read-private playlist-modify-public playlist-modify-private user-modify-playback-state user-read-playback-state

Step 02

Schedules & phases

Build as many schedules as you need (e.g. Weekday, Weekend, Festival). Each schedule has any number of named phases that run at times you pick.

Step 03

Moods — click one to search playlists or songs

Tap a mood to open the search panel. Toggle between Playlists and Songs, then click any result to add it to that mood's queue and play it.

Mood library — reorder, add, remove
